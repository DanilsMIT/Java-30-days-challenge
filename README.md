# Java-30-days-challenge

Proyecto inspirado en JavaScript30 challenge: colección de ejercicios en JavaScript puro enfocados en manipulación del DOM, eventos y lógica. Demuestra habilidades para crear interfaces dinámicas sin frameworks.

En este repositorio se implementan distintos ejercicios de la web:https://javascript30.com, que fortalecen habilidades clave como manipulación del DOM, manejo de eventos y lógica de programación, aplicando conceptos fundamentales para construir aplicaciones web dinámicas.
